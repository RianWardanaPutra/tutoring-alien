# Membuat page
Page berbeda dengan post. Jika post kita buat secara rutin, page berguna untuk menampilkan halaman statis yang tidak berubah, misalnya halaman Tentang atau halaman Portofolio. Secara default kita sudah memiliki 1 page yaitu halaman Beranda atau Home. Untuk menambahkan page, kita klik `Laman` &rarr; `Tambah Baru`

![new-page](/page/1.png)

Kita akan dihadapkan pilihan template untuk membuat page.

<Bleed>
![page-template](/page/2.png)
</Bleed>

Kita bisa memilih atau jika ingin kita bisa mulai dengan Halaman Kosong di sisi kiri. Tampilan untuk membuat halaman dengan Halaman Kosong sama persis dengan halaman untuk membuat post. Kita bisa mendesain halaman sesuka hati kita dengan blok-blok sehingga memudahkan dan tidak menuntut kita untuk mengerti pemrograman web.

Rangkuman langkah-langkah membuat Page jadinya:
1. Dari menu navigasi, arahkan pointer mouse ke Laman / Pages, kemudian klik `Tambah Baru`
1. Pilih dari template yang disediakan atau mulai dengan `Halaman Kosong`.
1. Edit laman sesuai kebutuhan seperti saat membuat post
1. Klik `Pratinjau` untuk melihat laman saat sudah diterbitkan atau langsung klik `Terbitkan` sama seperti saat membuat Post.
1. Laman sudah selesai diterbitkan di blog kita dengan link default sesuai judul laman.
1. Untuk Laman / Pages tidak masuk ke daftar post, sehingga kita harus mengatur link menuju laman tersebut dari pengaturan `Menu`

Selanjutnya kita akan belajar membuat menu / group.
