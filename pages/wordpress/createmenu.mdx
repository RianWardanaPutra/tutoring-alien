# Membuat group / menu

Group atau menu digunakan untuk navigasi di website kita. Seperti toolbar di bagian atas wp-admin dan menu navigasi di bagian sisi, menu atau group akan memudahkan pengunjung web kita untuk menjelajahi web kita.

Untuk membuat menu, dari menu navigasi kita akses Tampilan &rarr; Menu

![navigation](/menu/1.png)

Kita akan mendapatkan tampilan menu navigasi seperti ini

![edit-menu](/menu/2.png)

Selanjutnya kita klik tombol Buat Menu Baru dan kita akan dihadapkan dengan side bar seperti ini

![menu-name](/menu/3.png)

Kita bisa langsung memilih nama menu baru, kita isikan di bagian Nama Menu. Setelah itu kita memilih posisi menu kita. Ada beberapa posisi yang tersedia, yaitu Primary / utama, Menu Footer, dan Menu Tautan Sosial.

Menu utama berada di header web kita,

<Bleed>
![menu-header-view](/menu/4.png)
</Bleed>

Menu utama biasanya digunakan untuk navigasi di dalam web, sehingga memudahkan pengunjung dalam menelusuri web kita.

Menu footer sesuai namanya berada di footer,

<Bleed>
![menu-footer](/menu/5.png)
</Bleed>

Terdapat satu tombol kecil di bagian bawah (tertutup logo pensil) berisi nama website kita. Itu adalah posisi menu footer. Banyak website menggunakan menu footer untuk navigasi yang lebih kompleks, ada yang lebih memilih footer bersih dan hanya berisi akun media sosial. Kita bebas untuk mengisi menu footer kita, namun orang biasanya akan mencari akun sosial media kita di menu footer jika tidak terlihat di bagian awal web.

Sedangkan Menu Tautan Sosial akan muncul di suatu tempat tergantung tema yang digunakan. Misal di tema yang saya gunakan, Hever, Menu Tautan Sosial muncul di bawah Menu Utama

![social-menu](/menu/6.png)

Dapat dilihat terdapat 3 logo wordpress, itu adalah item yang ada di menu sosial di tema Hever. Menu sosial ini biasanya digunakan untuk meletakkan tombol share ke media sosial lain, atau meletakkan akun media sosial penulis web.

Mari kita kembali lagi. Untuk menambah menu, masukkan nama menu. Misalkan kita memasukkan nama menu Akun dan pilih lokasi di Menu Tautan Sosial.

![insert-menu-name](/menu/7.png)

Klik tombol Berikutnya.

Selanjutnya kita perlu menambahkan item yang akan dipasang di menu tersebut.

![add-item](/menu/8.png)

Klik Tambahkan Item untuk item baru lalu pilih laman yang akan dimasukkan ke dalam daftar Item

![add-item-2](/menu/9.png)

Atau kita juga bisa memasukkan item lain yang sesuai keinginan kita. Kita bisa memasukkan Pos khusus, Kategori, Tag, atau Link menuju web luar dengan Taut Tersuai.

![add-item-3](/menu/10.png)

Kita bisa mencampur item dalam satu menu, akan tetapi lebih memudahkan pengunjung website kita jika item-item yang ada dalam satu menu sejenis. Misalkan semua item adalah tautan media sosial atau tags.

![changes](/menu/11.png)

Terlihat telah ditambahkan menu dengan ikon Youtube di pojok kanan.
Selanjutnya, klik tombol Simpan Perubahan di pojok kiri atas.
Kita telah selesai menambahkan menu baru media sosial di website kita.

Rangkuman langkah-langkah membuat dan mengubah Menu adalah:

1. Dari menu navigasi, arahkan mouse / pointer ke Tampilan, lalu klik tombol Menu
1. Untuk membuat baru, klik tombol Buat Menu Baru, lalu masukkan nama menu dan posisi menu
1. Untuk mengubah menu, klik menu yang sesuai
1. Tambahkan atau hapus Item yang sesuai, lalu kilk tombol Simpan Perubahan.
1. Menu sudah selesai dibuat atau diubah.

Kini kita sudah memiliki website blog yang sudah bekerja. Kita tinggal mengatur tampilan lebih lanjut sesuai dengan keinginan kita. Pengaturan tema biasanya hanya dilakukan pertama kali dan lupakan. Kita tinggal mengatur laman baru dan post-post blog kita saja.
