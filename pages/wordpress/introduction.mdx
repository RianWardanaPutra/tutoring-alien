# Membuat Wordpress

Nah kita sampai di bagian utamanya, yaitu membuat blog wordpress. Perlu diingat kembali saya tidak dibayar untuk mengiklankan wordpress tetapi mengingat wordpress memiliki performa dan reputasi yang baik sehingga tutorial kali ini akan membahas wordpress secara seksama. Kita akan membahas beberapa hal, di antaranya:
1. Apa yang kita dapatkan dari wordpress
2. Cara mendaftar pada [Wordpress.com](https://wordpress.com) dan mendapat hosting gratis
3. Cara menginstall wordpress jika memiliki hosting sendiri
4. Mengakses laman admin
5. Cara memasang tema
6. Cara membuat post dan page (dan perbedaannya)
7. Cara membuat group atau menu

Kita langsung saja mulai dari "Apa saja yang didapatkan dari Wordpress"
