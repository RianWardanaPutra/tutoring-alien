# Apa saja yang didapatkan?
Nah apa yang didapatkan jika kita menggunakan wordpress? Yang pertama pasti kita akan mendapat kemudahan untuk membuat website hanya dengan beberapa langkah klik klik saja. Kita tidak perlu pusing memikirkan cara membuat web dari dasar menggunakan PHP atau bahasa pemrograman lain. Kita tidak perlu memikirkan cara membuat style di html kita karena semuanya tinggal drag-n-drop saja.
