# Content Management System

Belajar menulis web memang berat jika kita belajar html, css dan javascript dari awal. Tentu akan berguna jika kita ingin menjadi web programmer. Jika hanya ingin memulai menulis blog, ada cara yang lebih sederhana yaitu dengan menggunakan Content Management System atau CMS. CMS memberikan kemudahan bagi kita untuk mengatur database, halaman, post dan lain sebagainya yang kita perlukan dalam melakukan blogging dengan lebih mudah. Terdapat beberapa CMS yang terkenal, seperti:

Dengan menggunakan CMS, kita tidak perlu mengatur hal-hal yang sulit seperti backend, koneksi ke database, bahkan untuk urusan frontend kebanyakan CMS menyediakan fitur drag-n-drop untuk mengedit halaman web, sehingga kita tidak perlu pusing memikirkan elemen html, css, dan javascript.

Untuk selanjutnya kita akan belajar melakukan blogging menggunakan Wordpress. Wordpress bisa digunakan untuk membangun bukan hanya blog tetapi juga e-Commerce, dokumentasi, dan website pada umumnya. Selain itu, Wordpress juga sangat populer dibandingkan dengan CMS lainnya karena Wordpress sangat fleksibel dan mudah bagi pemula.
